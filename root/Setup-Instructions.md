#POLYCHAT
##SETUP INSTRUCTIONS
 

1.Backend setup
-cd backend(to navigate to the backend folder)
-npm install(to install the required dependencies)
-npm run dev(to start the backend server)

-.env file should be made inside the backend set up containing essential API keys
-For OpenAi(gpt 4o-mini)
  -Go to github
  -settings->developer settings
  -personal access tokes->tokens(classic)
  -generate new token
  -Copy the key
-For Gemini-2.5-flash
  -Go to google ai studio
  -Go to api keys
  -Create a api key for new project
  -Copy the key
-For Command-a-03-2025
  -Go to api keys
  -Get the trial api key
  -Copy the key

IF THE SERVER STARTS SUCCESSFULLY THEN THE USER MUST SEE:-
-Server running on port 3001-

2.Frontend setup
-Go to screen1.html
 -right click
 -click open with live server
-Select any model of choice
-Start your chat
-After pressing enter,it will switch to screen2.html
